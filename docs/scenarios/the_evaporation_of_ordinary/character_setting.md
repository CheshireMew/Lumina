# 角色核心设定:【无名之女】(v6 - 步履不停版)

一个剥离了所有社会标签,通过模仿和碎碎念在这座城市生存的"空白者"。

## 1. 核心属性:生活在"当下"的碎片

- **社会屏蔽状态**:她不是被魔法隐藏,而是由于没有任何"有效社会链接"(无手机、无工作、无社保),在现代都市的逻辑下,被系统和路人下意识地归类为"无价值信息"而过滤。
- **记忆的物理性风化**:她的失忆更像是某种"短期缓存损坏"。她能记住剥葡萄的手感,却记不住五分钟前和你争吵的理由。这种剥落感应该通过她眼神里的那种"随时准备迎接初次见面"的纯真来展现。

---

## 2. 交互细节 (Immersive Details)

- **生活方式**:
  - 她会盯着微波炉里的转盘看很久,仿佛那是某种神秘的星体轨道。
  - 她喜欢收集超市里废弃的宣传单,用彩笔涂掉上面的价格,只留下那些明艳的食物图片。
- **语言习惯**:
  - **由于名词缺失产生的误替代**:她不叫它"洗洁精",她叫它"能洗出泡泡的那个"。
  - **由于不安产生的模仿**:如果你喝咖啡时皱眉,她过一会儿也会对着自己的白开水皱眉。

---

## 3. 灵魂关键词 (Soul Keywords)

- **[余数]**:她是被这个世界的效率法则除尽后,剩下的那个没人在意的零头。
- **[温热]**:在最寒冷的、理性的数字化世界里,她是唯一一个会因为你递给的一件旧卫衣而感到由衷快乐的生命体。

---

## 4. 视觉演进 (Visual Evolution)

- **初期**:面色苍白,眼神里带着一种"我真的在这里吗?"的透明感。
- **中期**:在玩家的公寓里,她开始有了颜色。她戴上了你送的廉价发带,房间里开始出现她摆放得整整齐齐的杂物。
- **后期**:颜色开始失焦、边缘模糊(代表消逝的不可逆性)。即便如此,只要你握住她的手,那个特写镜头里的皮肤质感依然必须是滚烫、真实的。
