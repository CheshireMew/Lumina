# 平行剧本库:【多维视野】(Alternate Visions) 规划大纲

本文件夹通过三种完全不同的世界观,探索人与人之间连接的不同形式。

---

## 1. 【电子幽灵】(Digital Ghost) —— 赛博悬疑 / 数据人性

**核心悬念**:在一个被算法彻底统治的城市,一个"Bug"是否拥有生存权?

| 章节    | 标题           | 核心剧情                                                       | 情感关键词            |
| :------ | :------------- | :------------------------------------------------------------- | :-------------------- |
| 第 1 章 | **系统溢出**   | 便利店结账时偶遇正在"加载"的像素女孩,帮她躲避扫描。           | 突兀 / 这里的静电     |
| 第 2 章 | **离线港湾**   | 将她带回未联网的老楼,尝试用旧显示器稳定她的"分辨率"。         | 构筑 / 粗糙的触感     |
| 第 3 章 | **碎片拼贴**   | 她开始"下载"城市垃圾数据中的记忆,揭露主角被辞退背后的隐秘。   | 觉醒 / 虚假的真相     |
| 第 4 章 | **格式化预警** | 城市清理程序定位到了公寓,玩家必须在物理和逻辑层面同时战斗。   | 惊险 / 最后的带宽     |
| 第 5 章 | **归零或无限** | 结局分支:将她上传至核心系统作为病毒(自由)或让她随数据消散。 | 壮烈 / 无法找回的备份 |

---

## 2. 【镜像反转】(Mirror Psychology) —— 心理超现实 / 自我调和

**核心悬念**:当你的童年记忆化为实体出现在眼前,你是要拥抱它还是驱逐它?

| 章节    | 标题           | 核心剧情                                                   | 情感关键词            |
| :------ | :------------- | :--------------------------------------------------------- | :-------------------- |
| 第 1 章 | **重逢的谎言** | 合同上出现陌生的童年签名,家里出现了黄雨衣女孩。           | 恐惧 / 熟悉的陌生感   |
| 第 2 章 | **锁住的柜子** | 探索记忆中的旧宅,解锁关于那场"意外"的真实碎片。           | 压抑 / 深埋的木刺     |
| 第 3 章 | **痛觉共享**   | 你受到的情感打击会同步在女孩身上产生物理伤痕。             | 痛楚 / 无法逃避的镜像 |
| 第 4 章 | **模糊的成年** | 你开始流失成年后的逻辑,变得越来越像那个无助的孩子。       | 迷失 / 时空的逆转     |
| 第 5 章 | **整合或放逐** | 结局分支:接受这份痛苦(治愈并长大)或留在那个静止的梦里。 | 释怀 / 镜子里的拥抱   |

---

## 3. 【凡人奇迹】(Ordinary Miracle) —— 治愈系 / 慢生活 / 现实关怀

**核心悬念**:两个被生活磨得没脾气的普通人,如何重新找回对"明天"的耐心?

| 章节    | 标题             | 核心剧情                                                     | 情感关键词            |
| :------ | :--------------- | :----------------------------------------------------------- | :-------------------- |
| 第 1 章 | **猫与未接来电** | 天台晒被单时的尴尬偶遇,发现彼此都是"面试逃兵"。             | 尴尬 / 太阳的味道     |
| 第 2 章 | **面试盟约**     | 约定互相监督去参加一次面试。目标不是成功,而是"准时到场"。   | 挣扎 / 唯一的衬衫     |
| 第 3 章 | **夜市的灯火**   | 15 块钱的奢侈:在路边摊分吃一串烤肉带来的卑微幸福。          | 温暖 / 烟火气的慰藉   |
| 第 4 章 | **生活的回马枪** | 面试再次失败点燃了积压的焦虑,两人在雨中的楼道爆发无声争论。 | 崩溃 / 现实的重力     |
| 第 5 章 | **凡人的清晨**   | 结局分支:在附近找个普通干活维持生活或离开城市回老家。       | 平淡 / 未必完美的出口 |
