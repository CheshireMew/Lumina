# 第一章:雪中的微火 (Refined)

## 场景:旷野铁轨上的废弃车厢

**氛围**:刺骨的寒冷。窗外白雪皑皑,苍白的阳光透过结霜的玻璃。车厢内部是深色的皮革和锈蚀的黄铜,空气中有一股冷冽的金属味。

### 1. 这种寒冷是真实的 (Sensory Awakening)

- **感官描写**:
  - 视野:模糊的白影在晃动。
  - 触觉:脸颊贴在冰凉的皮革椅背上,呼吸时在空气中凝成白雾。
  - 交互:玩家尝试搓手取暖。这时,系统不应跳出"主线任务",而是显示一句话:_"如果再不找点热量,指尖就要失去知觉了。"_

### 2. 火种与影子 (Encounter)

- **发现**:车厢尽头有一丝火光。一个披着灰色军大衣的身影蜷缩在角落。
- **互动**:
  - 没有玉佩,没有宿命的对视。
  - 你走过去,她略带戒备地往阴影里缩了缩。
  - 火堆上架着一个脏兮兮的马口铁罐头盒,里面正咕嘟咕嘟煮着雪水。
- **AI 指令**:角色不说话。她只是默默地递给你一个破旧的杯子,眼神清冷,像是在观察一个即将冻死的野生动物。

### 3. 收音机的断音 (Deeper Immersion)

- **情节**:角落里摆着一台老式收音机,正顽强地发出沙沙的底噪。
- **碎片**:偶尔会漏出一秒钟的人类歌声,转瞬即逝。
- **情感点**:当你喝下一口热水,那种热量贯穿全身的瞬间,收音机里突然响起了一个温柔的音节。就在那一刻,你们的视线才第一次真正交汇。
- **沉浸式悬念**:你发现收音机的电池盒上刻着一行由于磨损而几乎消失的小字:_"致……最后的幸存者。"_
