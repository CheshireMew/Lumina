# 第一章:熄不灭的灯火 (v3 - Relatable)

## 场景:深夜 3 点的杂乱公寓

**氛围**:温暖但压抑。窗外雨声嘈杂,玻璃上凝聚着水雾。室内光线昏暗,只有一盏落地灯亮着。

### 1. 这种生活感是真实的 (Domestic Awakening)

- **感官描写**:
  - 声音:雨滴敲击铝合金窗框的节奏,老旧电冰箱轻微的嗡鸣声。
  - 触觉:棉质被单的质感,枕头残留的一点点温热。
  - 嗅觉:雨水的潮气,混合着不知名洗发水的清香。
- **交互**:玩家伸手去摸床头的手机,却发现那里只放着一瓶快要过期的感冒药。系统提示:_"头很沉,这里似乎不是你该在的地方。"_

### 2. 窗边的倒影 (Encounter)

- **发现**:一个少女背对着你坐在书桌前。她低着头,笔尖在纸上划动的声音在雨夜显得格外刺耳。
- **互动**:
  - 没有奇幻开场。你坐起身,床板发出的吱呀声让她停下了笔。
  - 她没有立刻回头,只是轻声说了一句:"你醒了?药在桌上,还是温的。"
- **AI 指令**:初期的语气要像普通的合租室友或者是相处很久的旧识,这种 **"伪造的亲昵感"** 是最大的悬念。

### 3. "门"外的裂口 (The First Crack)

- **情节**:你试图起身去洗手间或者推门离开。
- **反差**:当你握住门把手,感受到冰冷的金属触感时,你无意间瞥到了窗外——
- **视觉冲击**:雨幕中,远处的摩天大楼像乐高积木一样正在无声地瓦解,化为无数发光的粒子向上方那个漆黑的巨洞飘去。
- **情感点**:她走到你身后,轻轻按住你放在门把手上的手。她的手指冰凉,声音带着一丝颤抖:"别开门……求你,再留在这里陪我一个晚上。"
