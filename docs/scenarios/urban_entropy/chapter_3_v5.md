# 第三章:破碎的盛夏 (The Fractured Summer - v5)

## 场景:城市公园 / 海边,烈日下的午后

**氛围**:明亮得近乎刺眼的阳光。蝉鸣声排山倒海,空气中带着柏油马路蒸发的咸味。

### 1. 偷来的"幸福"

- **情节**:你拿着遣散补偿金,带她经历了你之前从未奢侈过的"假期"。你们去吃昂贵的冰激凌,去老旧的电影院看那种无聊的爱情片。
- **互动**:她对这一切都表现出一种极度的、近乎贪婪的热情。她会拉着你去看每一朵盛开的花,会为了在自动抓娃娃机里抓到一个丑陋的小熊而大笑。
- **画面感**:那个下午,你们坐在海边的长椅上。海风吹乱了她的头发。你第一次感觉到,即便没有了工作,没有了社会地位,只要握着这个人的手,你似乎也能在这片废墟上扎根。

### 2. 第一道"裂痕"

- **动作**:你试图和她聊起昨晚你们一起看的那部电影的结局。
- **对话**:
  - **你**:"那部电影的最后,男主角到底有没有回去找她?"
  - **她**:(笑容僵了一下,眼神里闪过一丝瞬间的空白):"电影……啊,是那部有很多云的电影吗?"
  - **由于心冷**:你看着她手中的那个小熊——那是你半小时前刚抓给她的。她正低头看着它,仿佛那是某种从天而降的、她从未见过的奇怪生物。
- **残酷的真相**:她开始频繁地问你:"我们今天中午吃的什么?"或者"你是谁?"。

### 3. "笔记"的背叛

- **情节**:回到家,你疯狂地翻开她的笔记本。
- **视觉细节**:你发现你之前写下的所有关于你的信息、她的名字、你们去过的地方,全部化为了模糊的灰色墨迹。更让你心惊的是,笔记本中间有几页被撕掉的痕迹——那是在你遇到她之前,别人为她写下的内容吗?
- **深度感悟**:_"她不是在遗忘。她是在被剥离。就像有一个无形的审计员,每过 24 小时就会清空她的缓存,确保这个沉重的社会系统里,没有任何冗余的、不产生价值的情感垃圾被保留。"_

### 4. 彻底的无力感

- **情境**:她注意到你在颤抖,走过来轻轻抱住你。她的体温是真实的,但她的眼神却像是一面干净得可怕的镜子,里面映照不出任何昨天留下的痕迹。
- **对话**:
  - **她**:"虽然我不记得你是谁了……但我的心跳告诉我,这里(指着胸口)现在跳得很疼。这就是你说的那种'难过'吗?"
- **悬念尾音**:你意识到,当你试图通过记忆来"培养"她时,你实际上是在给一个注定漏水的容器灌水。你灌得越猛,流失时的痛苦就越尖锐。
