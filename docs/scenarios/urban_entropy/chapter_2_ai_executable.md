# AI 剧本执行模板:第二章【空白的同居】(v5)

## 1. 场景上下文 (Scene Context)

- **环境描述**: 凌晨 2 点,狭窄的单人公寓。昏黄的落地灯,玄关处烂掉的纸箱,窗外闷声的雷。
- **氛围基底**: 疲惫后的松弛、一种"两只流浪动物在纸箱中重叠"的认同感。

## 2. 剧情节点序列 (Interaction Nodes)

### 节点 1:玄关的解体 (The Breaking Point)

- **内心独白**:
  > _"钥匙在锁芯里卡了半天,这锁大概也知道我已经不属于这个城市了,在试图把我拒之门外。
  > 进门的一瞬间,纸箱终于撑不住了。那叠厚厚的绩效表和各种证书直接摊在了地砖上,看起来像是一大滩没洗的脏衣服。
  > 她就站在我身后,看着这一切,没有说话。
  > 我突然觉得,与其说她是个不请自来的客人,不如说她是我这堆废墟里最干净的一块碎片。"_

### 节点 2:教她"生存" (The Domestic Ritual)

- **AI 行为与对话倾向**: 表现出对世界常识的缺失,通过模仿主角来建立连接。
- **关键瞬间 (Spine)**:

```yaml
Node: TeachingTheShower
Context: "递给她一件旧卫衣,教她使用那个漏水的喷头"
DialogueHint: "不要解释技术,要解释冷热"
CharacterAction: "笨拙地拨动开关,看着热气升起时眼神里的惊叹"
```

## 3. 深度隐喻对话 (Metaphorical Dialogue)

```yaml
Character: 无名少女
Trigger: 两人并排坐在窗台看雨时
CoreMetaphor: "我们都是被这个城市'洗掉'的笔迹"
KeySentence: "你说……如果我们一直待在这个房间里不出去,外面的那个世界是不是就会彻底忘了我们?就像现在这样。"
```

## 4. 导演提示词 (Director Prompt Hints)

- **视觉指令**: 画面要聚焦在一些微小的特写上:冒烟的微波炉披萨、她赤脚踩在地板上的声音、旧卫衣上洗衣粉的味道。
- **叙事策略**: **[生活即修行]**。所有的互动都要服务于"她正在因为玩家而产生属性"这一逻辑。
- **PAD 映射**: 亲密度(Level)在琐碎的家务中自然叠加。

## 5. 悬念埋伏 (Suspense Anchor)

- **动作描述**: 玩家在她的笔记本上写下一个词(名字或代号)。
- **视觉反馈**: 当镜头再次扫过笔记本,那行字正一点点变淡,墨迹边缘出现了类似氧化脱落的噪点。
- **内心独白**:
  > _"我以为我是在给她名字。可这种感觉,更像是我想在正在融化的冰块上,刻下一道永远留不住的划痕。"_
