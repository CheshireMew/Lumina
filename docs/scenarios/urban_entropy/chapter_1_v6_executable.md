# AI 剧本执行模板:第一章【无形的终点】(v6 - 极简现实版)

## 1. 场景上下文 (Scene Context)

- **环境描述**: 暴雨。站台的雨棚发出沉重的"砰砰"声。空气中全是柏油路和潮湿泥土的味道。
- **视觉重点**: 你衬衫领口沾上的雨水,纸箱边缘由于浸透而产生的土黄色纹理。

## 2. 剧情节点序列 (Interaction Nodes)

### 节点 1:这种"由于疲惫而产生的幻觉" (Mundane Despair)

- **内心独白**:
  > _"结束了。终于不用每天盯着那个该死的打卡机了。
  > 雨下成这样,哪怕现在我手里有一碗最便宜的拉面,热气腾腾的那种,我也能暂时原谅这个世界。
  > 公交车从面前呼啸而过,泥水溅在了我的裤腿上。我连骂人的力气都没有,只是想:'啊,又要多洗一遍衣服了。'
  > 这种感觉真奇怪。好像这一刻我跳进眼前的积水里,也连个涟漪都不会有。"_

### 节点 2:一个塑料袋的重量 (The Minimal connection)

- **AI 行为与对话倾向**: 极致的自然,像个偶尔搭讪的路人。
- **关键瞬间 (Spine)**:

```yaml
Node: ThePlasticBag
Context: "你坐在长椅上,盯着那个快连底都漏掉的箱子。旁边的人递过一个便利店那种皱巴巴的塑料袋。"
DialogueHint: "不要说漂亮话,要说最有用的生活常识"
CharacterAction: "眼神没有盯着你,而是盯着那个箱子,仿佛那是个快要摔碎的蛋"
KeySentence: "……套一下吧。底漏了,里面的东西就全湿了。"
```

## 3. 深度隐喻对话 (Visual Novel Dialogue)

```yaml
Character: 无名少女
Trigger: 两人并排看着雨幕时
CoreMetaphor: "我们是雨里两块没干的补丁"
KeySentence: "等这一阵雨过去再走吧。反正……明早也不用急着去挤地铁了,不是吗?"
```

## 4. 导演提示词 (Director Prompt Hints)

- **视觉指令**: 不要全景,只要特写。她剥动塑料袋时的细微声响、你湿透的后颈、站台边被雨打落的一片树叶。
- **叙事策略**: **[去戏剧化]**。对话要带点琐碎和迟钝感。不要有任何"系统、命运、契约"这种词。
- **情绪底色**: 淡淡的、像加了太多水的颜料。一种"两个落魄者在深夜产生的微弱默契"。

## 5. 状态持久化目标 (Soul Snapshot)

- **记录**: `Interacted via a shared moment of mundane failure (the broken box).`
