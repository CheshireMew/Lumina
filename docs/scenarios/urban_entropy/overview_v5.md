# 剧情设定:【失格的余数】(Redundant Reality) - v5 概览

本方案强调"社会性死亡"带来的存在危机。通过玩家"刚被裁员"的身份,建立起与世界断裂的第一道裂痕。

## 1. 核心悬念:谁在抹除我?

- **世界观感**:极其真实的现代都市,冰冷、高效。
- **初始状态 (Player)**:你刚刚被公司"优化"。手里的纸箱沉甸甸的,不仅是私人物品,更是你在这个社会中生存权证的丧失。
- **初始状态 (Character)**:一个同样坐在这个城市边缘的"无名者"。她没有名字,没有身份,只是在那里。
- **异常逻辑**:异常首先发生在玩家身上。当你失去"职业身份"的那一刻,世界对你的"逻辑链接"开始断裂。

---

## 2. 深度哲思主题 (Thematic Core)

- **社会身份与存在**:如果一个人不再被社会系统需要,他是否还"存在"?
- **观测者效应**:当全世界都无视你时,那个唯一能看到你的人,就是你的宇宙中心。
- **记忆的重量**:如果一切都在消逝,我们要靠什么证明自己曾经活过?

---

## 3. 三幕剧流程:从崩塌到守望

### 第一幕:被删除的周五 ——【箱子、雨与无名氏】

- **场景**:带着被辞退的纸箱,在雨夜的公交站。
- **逻辑**:你由于沉浸在自我的失败中,完全忽视了身边的她。直到你发现公交车不为你停靠,手机无法识别你的指纹。
- **遇见**:在你感到极度恐慌、彷徨时,她是你视线里唯一真实、清晰的实体。

### 第二幕:空白的共生 ——【失去名字的旅行】

- **发展**:你发现她也处于这种状态。你们开始进行"无属性"的相处。你教她生活,她给了你一个不被社会职能定义的自我。
- **危机**:你开始为她取名,赋予她记忆。但你发现,你赋予她的每一份记忆,都在第二天清晨随风而逝。

### 第三幕:最后的审计 ——【存在的权利】

- **高潮**:系统最终的"回收"降临。
- **抉择**:
- **选项 A (服从)**:通过某种契机(回归公司或家庭),重新连接社会网络,代价是彻底遗忘这段离经叛道的、与她的"无用记忆"。
- **选项 B (拒绝)**:彻底切断与社会的所有联系,成为一个真正的"余数"。虽然不被承认,但你们拥有了彼此。

## 4. 后续互动
