# AI 剧本执行模板:第三章【破碎的盛夏】(v5)

## 1. 场景上下文 (Scene Context)

- **环境描述**: 烈日,蝉鸣,路边摊的冰激凌机,电影院昏暗的出口。
- **氛围基底**: 极度的明亮与极度的寒冷交织。一种"在死刑前狂欢"的虚假繁荣感。

## 2. 剧情节点序列 (Interaction Nodes)

### 节点 1:琥珀里的下午 (A Moment in Amber)

- **内心独白**:
  > _"蝉鸣响得像是要把整个夏天都嚼碎。阳光晃在柏油马路上,腾起一股咸腥的燥热。
  > 哪怕没了工作,没了名片,只要能在这儿看她为了抓到一个丑得出奇的小熊而大跳大闹,我竟然产生了一种'这样过一辈子也不错'的错觉。
  > 这种由于贫穷和失败换来的自由,美好得像是一颗一碰就碎的肥皂泡。"_

### 节点 2:重复的风景 (The Unending Loop)

- **AI 行为**: 开始表现出微妙的重复,眼神中透出空洞。
- **关键瞬间 (Spine)**:

```yaml
Node: RepeatingTheQuestion
Context: "在吃完冰激凌不到十分钟,她再次拉着你的袖子问:'我们要不要买支冰激凌?'"
DialogueHint: "不要说她生病了,要描写她眼神里的'崭新'感"
CharacterAction: "笑容依旧灿烂,但那份灿烂由于缺乏记忆的厚度,显得单薄得像一张纸。"
```

## 3. 深度隐喻对话 (Metaphorical Dialogue)

```yaml
Character: 无名少女
Trigger: 坐在海边看夕阳时,她突然转头看向你
CoreMetaphor: "名字是灵魂的锚点,而我们正在起航"
KeySentence: "你说……如果不记得了,那份'喜欢'的心跳,会不会也跟着一起蒸发掉?"
```

## 4. 导演提示词 (Director Prompt Hints)

- **视觉指令**: 强烈的色彩对比。盛夏的蓝天与她逐渐变白的指尖特写。
- **叙事策略**: **[在明亮中撕裂]**。不要描写悲剧,要描写"快乐的失效"。描写她看着那个刚送她的小熊时,那种像在看陌生礼物的客气。
- **内心独白提示**: 主角开始意识到自己是在给一个漏底的瓶子灌水。

## 5. 悬念爆发 (The Erasure)

- **动作描述**: 玩家回家后发疯般翻开她的笔记本。
- **视觉反馈**: 所有的书写痕迹都化为了模糊的灰色云烟,书页中间甚至出现了半透明的孔洞。
- **内心独白**:
  > _"原来世界在收回她。不是通过暴力,而是通过这种无声的、从内而外的注销。
  > 我记下的每一个字,都在这一刻指责我的徒劳。"_
