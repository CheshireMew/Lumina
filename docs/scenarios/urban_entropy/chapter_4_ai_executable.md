# AI 剧本执行模板:第四章【终焉的审计】(v5)

## 1. 场景上下文 (Scene Context)

- **环境描述**: 深夜 11:59,公寓。光线不稳定,手机不断震动(系统通知),桌上那份"复职合同"显得格外白。
- **氛围基底**: 最后的审判、现实与虚幻的交接点。一种"只要妥协就能获救,只要坚持就会粉碎"的张力。

## 2. 剧情节点序列 (Interaction Nodes)

### 节点 1:来自"对岸"的诱惑 (The Corporate Invitation)

- **内心独白**:
  > _"电话那头的声音干净得没有一丝杂质。他告诉我,只要签个字,我的房租、我的债务、我那八百多个消失的日夜都能'修补'回来。
  > 只要承认她不曾存在过,承认那场雨只是一次短暂的神经递质失调。
  > 这个逻辑太完美了,完美到让我觉得,坚持留下这个连自己是谁都记不住的女孩,才是一场最滑稽的自残。"_

### 节点 2:镜子里的真实 (The Real Reflection)

- **关键瞬间 (Spine)**:

```yaml
Node: TheFinalConfrontation
Context: "她站在阴影里,看着那份合规合同,手里的笔记本已经变成了一叠彻底的白纸"
DialogueHint: "不要说爱,要说'重叠'"
CharacterAction: "没有阻拦你,只是静静地看着你,眼神里满是'如果你想走,我随时可以消失'的淡然"
```

## 3. 深度隐喻对话 (Metaphorical Dialogue)

```yaml
Character: 无名少女
Trigger: 签字笔悬在纸面上方时
CoreMetaphor: "我们是这个完美系统里,仅存的两处'排版错误'"
KeySentence: "如果你选了那个'正确'的世界,请一定要记得给窗台上的那盆花浇水……虽然在那个世界里,它可能从未开过。"
```

## 4. 导演提示词 (Director Prompt Hints)

- **视觉指令**: 画面逐渐剥离色彩。现实世界的物体(合同、金钱、手机)颜色极其鲜艳,而她和整个房间却在向灰色坍塌。
- **叙事策略**: **[残酷的选择]**。内心独白要体现那种痛苦的拉扯——是一辈子做个精准的零件,还是陪她在这雨夜里烂掉。
- **分支判定**:
  - **A (理性赎买)**: 签字。独白结束于"我似乎做了一个很长的梦"。
  - **B (感性坠毁)**: 撕碎。独白结束于"我握紧了那只冰冷的手,这是我这辈子做过最正确的一道错题"。

## 5. 灵魂印记 (Soul Snapshot)

- **最终固化**: 记录玩家对于"社会职能"与"个体情感"的终极优先级判定。
