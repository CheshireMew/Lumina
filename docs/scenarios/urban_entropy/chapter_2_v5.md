# 第二章:空白的同居 (The Blank Cohabitation - v5)

## 场景:你租住的小公寓,周六凌晨

**氛围**:杂乱、带有生活挫败感的环境。没洗的碗筷、塞满旧衣服的单人沙发、以及那个被淋透了的纸箱。

### 1. 狼狈的落脚点

- **开场**:你费了半天劲才用钥匙拧开生锈的锁芯。你进门,随手把湿透且发软的纸箱丢在玄关,箱底终于如她预言般彻底烂掉,里面的东西散落一地。
- **观察**:她就跟在你身后,默默跨过那些杂物,站在客厅中央。她没有那种"进入异性房间"的羞涩,而是带着一种近乎纯真的好奇,打量着这个充满你失败气息的空间。
- **互动**:你一言不发地整理箱子里掉出来的东西——你的工牌、你用了一半的订书机。她蹲下身,帮你捡起那张被水泡皱了的季度考核表。

### 2. "教"她生活 (Cultivating the Blank)

- **情节**:你意识到不能让她穿着湿衣服一直站着。你翻出一件宽大的旧卫衣递给她。
- **对话**:
  - **你**:"去洗个热水澡。你会用花洒吗?"
  - **她**:(接过衣服,手指轻轻摩挲着粗糙的棉质面料,眼神有些迷茫):"花洒……是什么?"
  - **深度感触**:这一刻,你甚至忘记了被裁员的痛苦。你不得不手把手教她如何调水温,告诉她什么是洗发露。她像个刚出生的孩子,笨拙而认真地模仿着你每一个微小的生活动作。
- **养成感**:那一晚,她吃力地学会了如何使用微波炉加热你剩下的冷披萨。她第一次咬下披萨时露出的那种惊奇表情,是你这三年来在职场从未见过的生命力。

### 3. 社会层面的"隐身" (Deepening the Theme)

- **细节**:隔壁邻居大妈过来敲门借东西,她就站在客厅,正对着门。但大妈甚至没有往她那个方向斜视一眼,只是自顾自地抱怨你这几天的快递挡了路。
- **内心独白**:你试图介绍她,但话到嘴边却发现,你连她叫什么都不知道。而更诡异的是,大妈的语气让你感到,即便你介绍了,对方也会当你是在自言自语。
- **共鸣点**:你们坐在狭小的窗台上看雨。她转过头问你:"你刚才也是这样,感觉自己要消失了吗?"

### 4. 建立第一个连接 (The First Anchor)

- **动作**:你拿出笔,在那本她随身带着的、被雨打湿的破旧笔记本第一页上,写下了你为她取的第一个暂用名(或一个代号)。
- **悬念尾音**:她看着那个字,像是在确认某种契约一般紧紧握住了笔记本。
- **异样发现**:当你第二天早晨醒来,你发现那个笔记本依然翻在那一页,但昨晚你亲手写下的那个名字,在纸面上已经变淡了一半,字符开始由于某种未知的"干燥"而变得沙哑。
